(static) fill(arrayToFill, symbol, startIndex, endIndex) → {Array}

Заменяет элементы массива переданными символами.
Parameters:
Name Type Default Description
arrayToFill Array.<number> массив, который будем обрабатывать.
symbol String символ (или строка) на которую будем заменять элементы массива.
startIndex Number 0 начальный индекс элемента массива, который будем заменять.
endIndex Number 0 конечный индекс элемента массива, который будем заменять.

Returns:
- итоговый массив, в котором заменены элементы.

Example
fill([1, 2, 3], 'x');

// -> [2, '*', '*', 8]
